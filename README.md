# 🌍 Climate News Website

A responsive website about climate change created using HTML, CSS, and JavaScript.

## About

This project was developed as part of a university web development course.

The website introduces environmental issues and climate change through a clean and easy-to-read layout.

## Technologies

* HTML5
* CSS3
* JavaScript

## Features

* Responsive layout
* Climate news sections
* Image-based design
* Simple navigation
